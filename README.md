Retcon. A server's statistics logger. 

## Requirements

* Java 8 or later
* Spigot, Paper, or something great for MC Server
* EssentialsX
* *PATIENCE*

## Target to logging

* Command
* Player's balance (from EssentialsX)

More features to be added.

## Command

* No command implemented. There's a command `/retcon` for dummy.
